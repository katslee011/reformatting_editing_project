Summary of Changes: Document 1

For my first document, I chose to reformat and edit an essay I wrote about the discourse of classical clarinetists. I chose this document because I wanted to format a document with a visual, which lets me practice writing alt text. To format this document, I used Markdown.

Reformatting the Document with Markdown

Markdown is a fairly simple markup language. To create paragraphs, separate them with blank lines; to create headings, use hashes; to bold or italicize text, use asterisks. My document didn't include any bold text, so all I had to do was add asterisks where the original document had italics.

The original "The Clarinetist Discourse" didn't have any headings, which worked well enough when it was an essay, but worked less well when converted into Markdown. When I added headings as part of my editing process, I used hashtags to create heading level 2s, which differntiated them from the title heading level 1.

One of the advantages of using markup languages instead of a word processor like Google Docs, which was where the original document was written in, is markup languages make it easy to add alt text. This document included one image, a photo of the back of a product catalog, so I added alt text accordingly. Following what I know about writing good alt text, I kept it short while carefully considering the context; right before the image is a length image description, reducing what information need to include in the alt text.

Finally, the references section included many URLs, which I thought would be better if they were hyperlinked. I did so using square brackets and parenthesis, which is how Markdown recognizes hyperlinked text.
